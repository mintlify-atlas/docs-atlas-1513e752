---
title: "List Orders"
api: "GET /api/purchase"
description: "Retrieve all purchase orders"
---

Retrieves a list of all purchase orders in the system.

<Warning>
  This endpoint returns all purchases without filtering. For user-specific purchases, use the `/api/purchase/user-purchases` or `/api/purchase/my-purchases` endpoints.
</Warning>

## Authentication

No authentication required for this endpoint.

## Response

Returns an array of purchase objects.

<ResponseField name="id" type="integer">
  The unique identifier for the purchase
</ResponseField>

<ResponseField name="user" type="object">
  The user who created the purchase
  
  <ResponseField name="user.id" type="integer">
    User ID
  </ResponseField>
  
  <ResponseField name="user.name" type="string">
    User's name
  </ResponseField>
  
  <ResponseField name="user.email" type="string">
    User's email address
  </ResponseField>
</ResponseField>

<ResponseField name="cart" type="object">
  The cart associated with this purchase
  
  <ResponseField name="cart.id" type="integer">
    Cart ID
  </ResponseField>
  
  <ResponseField name="cart.finalPrice" type="float">
    Total price of all items in the cart
  </ResponseField>
  
  <ResponseField name="cart.items" type="array">
    List of items in the cart
    
    <ResponseField name="cart.items[].id" type="integer">
      Cart item ID
    </ResponseField>
    
    <ResponseField name="cart.items[].quantity" type="integer">
      Quantity of the product
    </ResponseField>
    
    <ResponseField name="cart.items[].product" type="object">
      Product details
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="date" type="string">
  ISO 8601 timestamp when the purchase was created
</ResponseField>

<ResponseField name="reservationTime" type="string">
  ISO 8601 timestamp for the reservation expiration
</ResponseField>

<ResponseField name="direction" type="string">
  Shipping address for the order
</ResponseField>

<ResponseField name="status" type="string">
  Purchase status. Possible values: `PENDING`, `CONFIRMED`, `CANCELLED`
</ResponseField>

## Example Request

```bash
curl https://api.example.com/api/purchase
```
