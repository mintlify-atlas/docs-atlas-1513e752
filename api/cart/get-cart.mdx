---
title: "Get Cart"
api: "GET /cart/{id}"
description: "Retrieve a shopping cart by its ID"
---

## Authentication

This endpoint requires a valid JWT token in the Authorization header.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the cart
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  Example: `Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`
</ParamField>

## Response

<ResponseField name="id" type="integer">
  Unique identifier of the cart
</ResponseField>

<ResponseField name="finalPrice" type="float">
  Total price of all items in the cart
</ResponseField>

<ResponseField name="items" type="array">
  Array of cart items
  
  <ResponseField name="id" type="integer">
    Unique identifier of the cart item
  </ResponseField>
  
  <ResponseField name="quantity" type="integer">
    Quantity of the product in the cart
  </ResponseField>
  
  <ResponseField name="product" type="object">
    Product details
    
    <ResponseField name="id" type="integer">
      Product ID
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Product title
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    
    <ResponseField name="price" type="float">
      Product price per unit
    </ResponseField>
    
    <ResponseField name="stock" type="integer">
      Available stock
    </ResponseField>
    
    <ResponseField name="active" type="boolean">
      Whether the product is active
    </ResponseField>
    
    <ResponseField name="mediaSrc" type="array">
      Array of product image URLs
    </ResponseField>
  </ResponseField>
</ResponseField>

## Error Responses

- **401 Unauthorized**: User session is not active or token is invalid
- **404 Not Found**: Cart with the specified ID does not exist

<RequestExample>

```bash cURL
curl -X GET "https://api.example.com/cart/123" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/cart/123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});

const cart = await response.json();
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
}

response = requests.get('https://api.example.com/cart/123', headers=headers)
cart = response.json()
```

</RequestExample>
