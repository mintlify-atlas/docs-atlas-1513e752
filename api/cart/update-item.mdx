---
title: "Update Cart Item"
api: "PATCH /cart/items/{id}"
description: "Update the quantity of an item in the shopping cart"
---

<Warning>
  **This endpoint is not currently implemented in the API.**
  
  The current implementation uses a full cart replacement model. To modify cart items, you need to create a new cart with the updated items using the [Create Cart](/api/cart/add-item) endpoint.
</Warning>

## Alternative Approach

To update cart items with the current API:

1. [Get the current cart](/api/cart/get-cart) to retrieve all items
2. Modify the items array in your application
3. [Create a new cart](/api/cart/add-item) with the updated items
4. Optionally [delete the old cart](/api/cart/clear-cart)
