---
title: Password Reset
---

The password reset flow consists of three steps:

1. **Request Reset** - Send a reset token to the user's email
2. **Validate Token** - Verify the reset token is valid
3. **Change Password** - Update the password using the validated token

---

## Request Password Reset

<api>POST /password/request</api>

Sends a password reset token to the user's email address.

### Request Body

<ParamField body="email" type="string" required>
  User's email address
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message indicating the reset code was sent
</ResponseField>

### Example Request

```bash
curl -X POST https://api.example.com/password/request \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com"
  }'
```

### Example Response

```json
{
  "message": "Se ha enviado un correo con el código de recuperación"
}
```

### Error Responses

<ResponseField name="404 Not Found">
  Returned when the email doesn't belong to any account
  
  ```json
  {
    "error": "El email no pertenece a ninguna cuenta"
  }
  ```
</ResponseField>

---

## Validate Token

<api>POST /password/validate</api>

Validates that a password reset token is correct and still valid.

### Request Body

<ParamField body="email" type="string" required>
  User's email address
</ParamField>

<ParamField body="token" type="string" required>
  Reset token received via email
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message indicating the token is valid
</ResponseField>

### Example Request

```bash
curl -X POST https://api.example.com/password/validate \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "token": "ABC123XYZ"
  }'
```

### Example Response

```json
{
  "message": "Token válido"
}
```

### Error Responses

<ResponseField name="400 Bad Request">
  Returned when the token is incorrect or expired
  
  ```json
  {
    "error": "El token enviado no es correcto"
  }
  ```
</ResponseField>

---

## Change Password

<api>POST /password/change</api>

Changes the user's password using a validated reset token.

### Request Body

<ParamField body="email" type="string" required>
  User's email address
</ParamField>

<ParamField body="token" type="string" required>
  Reset token received via email
</ParamField>

<ParamField body="newPassword" type="string" required>
  New password for the account
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message indicating the password was changed
</ResponseField>

### Example Request

```bash
curl -X POST https://api.example.com/password/change \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "token": "ABC123XYZ",
    "newPassword": "newSecurePassword456"
  }'
```

### Example Response

```json
{
  "message": "Contraseña cambiada exitosamente"
}
```

### Error Responses

<ResponseField name="400 Bad Request">
  Returned when the token is incorrect or expired
  
  ```json
  {
    "error": "El token enviado no es correcto"
  }
  ```
</ResponseField>
