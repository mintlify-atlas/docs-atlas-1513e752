---
title: 'List Products'
api: 'GET /api/products'
description: 'Retrieve a paginated list of all active products'
---

Returns a paginated collection of all active products in the catalog. Only products with `active=true` are included in the response.

## Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number (zero-indexed)
</ParamField>

<ParamField query="size" type="integer" default="20">
  Number of items per page
</ParamField>

<ParamField query="sort" type="string">
  Sort field and direction (e.g., `price,asc` or `title,desc`)
</ParamField>

Returns a paginated response containing an array of product objects.

## Implementation Notes

- Only products with `active=true` are returned (see ProductController.java:64)
- Results are filtered after pagination, so page size may vary
- Default pagination uses Spring Data defaults (typically 20 items per page)
- Authentication required via JWT token
