---
title: 'Filter Products'
api: 'POST /api/products/filter'
description: 'Filter products by price, brand, category with sorting and pagination'
---

Filter and sort products based on multiple criteria including price range, brand, and category. Supports both single and multiple brand/category filtering.

## Request Body

<ParamField body="priceMin" type="float">
  Minimum price threshold
</ParamField>

<ParamField body="priceMax" type="float">
  Maximum price threshold
</ParamField>

<ParamField body="brandCode" type="integer">
  Single brand code to filter by (deprecated, use brandCodes instead)
</ParamField>

<ParamField body="brandCodes" type="array">
  Array of brand codes to filter by (supports multiple brands)
</ParamField>

<ParamField body="categoryCode" type="integer">
  Single category code to filter by (deprecated, use categoryCodes instead)
</ParamField>

<ParamField body="categoryCodes" type="array">
  Array of category codes to filter by (supports multiple categories)
</ParamField>

<ParamField body="sortBy" type="string">
  Sort field: `price` or `relevance` (relevance sorts by calification)
</ParamField>

<ParamField body="sortOrder" type="string">
  Sort direction: `asc` or `desc`
</ParamField>

<ParamField body="page" type="integer" default="0">
  Page number (zero-indexed)
</ParamField>

<ParamField body="size" type="integer" default="20">
  Number of items per page
</ParamField>

## Filter Logic

### Price Filtering
- Products with `price < priceMin` are excluded (ProductController.java:229)
- Products with `price > priceMax` are excluded (ProductController.java:232)
- Products with null prices are excluded if price filters are applied

### Brand Filtering
- `brandCodes` array takes precedence over single `brandCode` (ProductController.java:236)
- Products must match at least one brand code in the array
- Products without a brand are excluded

### Category Filtering
- `categoryCodes` array takes precedence over single `categoryCode` (ProductController.java:246)
- Products must have at least one category matching any code in the array
- Products without categories are excluded

### Sorting
- **price**: Sort by product price (ProductController.java:260-265)
- **relevance**: Sort by calification/rating (ProductController.java:266-271)
- Default order is ascending unless `sortOrder=desc`

### Active Products
- Only products with `active=true` are included (ProductController.java:225)

## Pagination
- Default page: 0
- Default size: 20
- Manual pagination applied after filtering and sorting (ProductController.java:275-280)
